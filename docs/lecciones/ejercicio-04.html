### Encuentre el error

Las siguientes líneas de código dan error, o no hacen lo que se supone que tienen que hacer. Copialas, pegalas, ejecutalas, y corregilas.

1) `library(lmerTest)`
_Si no instalé antes la librería lmerTest, no la voy a poder cargar y no va a funcionar._

2) `factor(datos$session, labels="1,2,3,4") #suponemos que los datos están en el data frame llamado "datos"`
_En vez de "labels", habría que indicarle con "levels". Además, hay que usar la función c(). Sería levels=c(1,2,3,4)_

3) `datosNoEjemplo$task %>% group_by(task)`
_El `group_by` es aplicable a data frames, no a un vector columna de un dataframe. `datosNoEjemplo %>% group_by(task)` no da error, pero aplicarle el `group_by` a la columna `task`de ese data frame si._

4) `datosNoEjemplo %>% group_by(task) %>% summarise(mean)`
_`summarise` (o `summarize`) admiten una (o mas) funciones aplicadas a alguna columna (o columnas) en particular. Aquí se le pide que calcule la media, pero... ¿a cuál de todas las columnas que tiene el data frame?
